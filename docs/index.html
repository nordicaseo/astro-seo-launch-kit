📋 The checklist

Every launch-critical SEO check — sitemap, robots, canonical, metadata, Open Graph, structured data, Core Web Vitals, images, redirects, accessibility, security headers.

🤖 The auditor

A zero-config CLI that crawls a live URL and auto-verifies 122 of the checks, with optional Core Web Vitals via PageSpeed Insights.

🖥️ The dashboard

A small login dashboard that runs the audit in the browser and shows a pass/fail score for the whole site.

🧠 The agent skill

A Claude skill so an AI agent can run the checklist and fix the issues in your Astro codebase until launch-perfect.

Audit any Astro site in one command

npx --yes degit nordicaseo/astro-seo-launch-kit seo-kit
cd seo-kit && npm install
node audit/cli.mjs https://your-site.com --json --html
